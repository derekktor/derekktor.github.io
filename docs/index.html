Neural Network Supervised vs Unsupervised Learning Regression Classification template
  • as any machine learning model, neural network gets input(age, education, profession) and then outputs a value(salary) to predict the input
  • has input layer, output layer and multiple hidden layers
  • each layer contains multiple neurons
  • at each layer, neurons are multiplied by weights and summed together with a bias and transformed by a activation function, specific to the layer
  • weights are unique to each neuron and at input layer, they're assigned arbitrarily
  • bias is unique for each layer, and depends on the activation function used
  • activation function is specific to each layer and transforms the sum into a value that can be used to decide whether if the neuron at current layer should be passed further(activated) or not
  • uses gradient descent to optimize the result
  • gradient descent uses the slope extracted from the graph of loss function to find the optimal value(minimum value for loss function)
  • loss function is used to calculate how far away is the prediction from the actual representation
  • neural network performs calculation iteratively to find the optimal result, by iterating cycles of forward-propagation and backward-propagation
  • forward-propagation is the cycle of calculating the end result by multiplying neurons with their weights and summing them up and then transforming according to their activation functions in order to get the prediction
  • backward-propagation happens for the first time only after a forward-propagation takes place
  • backward-propagation calculates error(from loss function) and then gains an answer to in which direction each weight should be changed to in order to minimize the overall error
  • Supervised learning uses labeled data while unsupervised learning uses unlabeled data
  • Supervised Learning
    • Regression
    • Classification
  • Unsupervised Learning
    • Clustering
    • Association
    • Dimensionality Reduction
  • Regression is when you want to find a relationship between dependent variables(predictors, Time of the day) and independent variables(target, Crowdness in the bus)
  • Types
    • Linear Regression
    • Logistic Regression
    • Polynomial Regression
  • assigns data into categories
  • Algorithms:
    • Linear classifiers
    • Support Vector Machines
    • Decision Trees
    • Random Forest
